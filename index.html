Osp-parser

A simple packet parser for Optin Kft's OSP (http://www.optin.hu/products/optin-sensor-protocol).

View on GitHub Download .zip Download .tar.gz

Welcome to osp-parser home page.

Feel free to contact me if you have an idea!

For install this package, simply type:

$ npm install osp-parser --save

Features

  • Parse raw OSP packages to editable objects
  • With plugins, you easily can parse incoming package data by the type of the package
